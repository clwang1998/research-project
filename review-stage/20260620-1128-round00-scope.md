# Round 00 Scope: Copilot Opus Reviewer Setup

## Modification Scope

- Set Codex as the default worker/executor for this repository.
- Set VS Code Copilot Chat with Opus as the read-only external reviewer.
- Kept Copilot CLI out of the default executor path to avoid two agents editing
  the same worktree.
- Added a neutral reviewer prompt wrapper at `scripts/build_reviewer_prompt.py`.
- Changed the legacy prompt builder defaults to target GitHub Copilot Opus and
  Codex.
- Documented manual MCP/prompt handoff and review archive naming.

## Reviewer Status

Copilot CLI sees the workspace `manual_review` MCP server, but this Codex
runtime does not expose `mcp__manual_review__review` as a callable tool. The
generated prompt is available at `tmp/copilot_opus_review_prompt.md` for manual
handoff to VS Code Copilot Chat with Opus selected.
