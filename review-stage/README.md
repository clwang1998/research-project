# Review Stage

Store every external reviewer round here when the task uses Copilot Opus as the
read-only reviewer.

Recommended file names:

```text
YYYYMMDD-HHMM-roundNN-copilot-opus.md
YYYYMMDD-HHMM-roundNN-scope.md
```

Each reviewer response file should contain the full Copilot Opus review. Each
scope file should summarize what Codex changed in response, what validation ran,
and which findings remain accepted risk.
